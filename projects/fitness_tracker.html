Project 2: Tracking Barbell Exercises

This mini Master’s project focuses on analyzing wearable sensor data to classify barbell exercises, count repetitions, and detect improper exercise form. Data from accelerometer and gyroscope sensors in Fitbit bands worn during gym workouts is processed through a robust pipeline including outlier removal, smoothing, temporal aggregation, frequency analysis.

Feature engineering with PCA and clustering. Machine learning models, mainly Random Forest, achieve 98.5% accuracy for exercise classification and form detection, and 95% accuracy for repetition counting. The project’s well-organized structure includes raw and processed data, notebooks for analysis, trained models, and visualization scripts. It references the book “Machine Learning for the Quantified Self” and builds upon research into context-aware strength training applications. Future work aims to expand datasets, integrate additional sensors, and enable real-time feedback via smartwatches. This project demonstrates advanced use of Python libraries such as pandas, numpy, scipy, scikit-learn, matplotlib, and seaborn for a sophisticated AI-powered fitness tracking system.

Key highlights include:

  • Accelerometer and gyroscope data collection using Meta Motion sensors
  • Data preprocessing: outlier removal, smoothing, temporal aggregation, frequency analysis
  • Feature engineering with PCA, clustering, and scalar motion magnitudes
  • Machine learning models for exercise classification (98.5% accuracy), repetition counting (~95%), and form detection (98.5%)
  • Organized project structure: data versions, notebooks, trained models, and visualization scripts
  • Integration of advanced Python libraries such as pandas, scikit-learn, matplotlib, and seaborn
  • Inspired by research on context-aware strength training applications
  • Future-ready for real-time tracking and expanded sensor integration

Project 3: Movie Recommendation System

This project is a **content-based Movie Recommendation System** that suggests the top-5 movies similar to a user’s input. It combines **machine learning (cosine similarity on text embeddings)** with real-time **TMDB API integration** to fetch posters and movie details.

Users can type a movie name, get intelligent suggestions (via fuzzy matching), and instantly see recommended movies with their **posters, release year, and overview**. The system is deployed as a **Flask web application** and integrated with Hugging Face Hub for model + dataset hosting.

Key highlights include:

  • Dataset of 5,000+ movies with tags built from title, genres, keywords, cast, crew
  • Feature engineering with TF-IDF embeddings and cosine similarity
  • Fuzzy matching for better handling of spelling variations
  • Live integration with TMDB API for fetching posters and metadata
  • Trending movies section for user engagement
  • Deployed Flask app with Hugging Face Spaces (Gunicorn server)
